# Kit éditorial premium immobilier (vendeurs)

Ce document fournit un cadre complet prêt à intégrer pour un écosystème immobilier local premium : header global, footer global, pages clés (Accueil, Estimation, Financement, Localité, Contact, Blog), CTA et maillage interne.

---

## Header global

### Objectif
Créer une première impression premium, locale et rassurante tout en orientant rapidement vers l’estimation.

### Structure recommandée
- **Logo (gauche)** : `NOM_MARQUE Immobilier`
- **Navigation (centre/droite)** :
  - Accueil
  - Estimation
  - Financement
  - Villes / Localités
  - Blog
  - Contact
- **CTA visible** : `Estimer mon bien`

### Micro-copy header
- Sur desktop : `Estimation locale, confidentielle et sans engagement`
- Sur mobile menu : `Votre projet mérite une estimation juste.`

### CTA header
- Principal : `Estimer mon bien`
- Variante : `Demander mon estimation`

### UX
- Header sticky discret
- Fond ivoire translucide + bordure fine
- Ombre légère au scroll
